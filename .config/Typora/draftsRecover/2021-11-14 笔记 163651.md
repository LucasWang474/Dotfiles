## 端口号

## 修改JS代码一定要重启Node服务

## 端口占用错误以及解决
```
{
  code: 'EADDRINUSE',
  errno: 'EADDRINUSE',
  syscall: 'listen',
  address: '::',
  port: 80
}
```
解决方法:
* 停止之前占用该端口的服务 (ctrl + c)
* 换一个端口启动服务
